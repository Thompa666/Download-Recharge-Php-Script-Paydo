Setup Guide

1. Upload all upload foler files to Hosting via FTP or File Manager

2. Now, time to create a database. Check Video

3. After that, Go to config files into "config" folder and set config.php and setup_config.php with database details.
into admin folder link.php and into admin folder > config folder > config.php
into distributor folder link.php and distributor > config folder > config.php
into cron folder > config.php

4. Now run install file like yourdomainname.com/install.php

5. Then Set with your website Details

6. Go to admin panel yourdomainname.com/admin login and set Recharge API key, Payment Key, SMS Key( support 2.1).

Now Enjoy!!! For Help Email - support@zetusolutions.in

Please note to update recharge status we need to set cron. so please run all cron folder files with cronjob from cpanel
